# ゲーム のタイトル
逆襲！エイリアン！
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターこうかとんをキーボードで操作し、敵機から放たれる爆弾から逃げたり撃ち落としたりしながら敵をを倒して生き残るゲーム

## ゲームの実装
### 共通基本機能
* 主人公キャラクターに関するクラス
* 爆弾に関するクラス
* 敵機に関するクラス
* スコアに関するクラス
* シールドに関するクラス
* 爆発に関するクラス
* 弾幕に関するクラス
* ビームに関するクラス
### 担当追加機能
* 残機システム(HP減少)
